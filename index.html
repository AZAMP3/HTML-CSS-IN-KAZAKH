HTML

HTML дегеніміз не?

HTML (HyperText Markup Language) - бұл веб-беттерді құруға арналған стандартты белгілеу тілі. HTML және CSS оқуды бастағандар үшін. Мұнда сіз барлық HTML тегтері мен атрибуттарының сипаттамасын, CSS қасиеттері мен мәндерінің сипаттамасын, көптеген код мысалдарын, сондай-ақ тиімді оқытуға арналған бейнелер мен тапсырмаларды таба аласыз.

  • B

  • <base>: основной URL страницы
  • <blockquote>: блок цитаты

  • C
  • <canvas>: область для рисования
  • <caption>: заголовок таблицы
  • <cite>: источник цитаты

  • <col>: столбец в таблице

  • D
  • <datalist>: выпадающий список для тега input
  • <dd>: элемент списка с описанием термина
  • <del>: удаленный текст
  • <details>: нативный "аккордеон"
  • <dfn>: выделение термина
  • <div>: блочный контейнер
  • <dl>: список с объяснением

  • <h4>: заголовок четвертого уровня

  • <head>: хранит информацию о документе
  • <header>: "шапка"
  • <hr>: горизонтальный разделитель

  • <i>: курсивный текст
  • <iframe>: встраивание страниц

  • L
  • <label>: ярлык элемента формы
  • <legend>: заголовок группы формы
  • <li>: элемент списка
  • <link>: подключение файлов стилей

  • <main>: основное содержание страницы
  • <map>: карта изображения
  • <mark>: подсвечивание текста

  • N

  • <noscript>: если скрипты отключены

  • <output>: результат вычислений

  • <p>: параграф

  • <progress>: индикатор выполнения

  • <s>: зачеркнутый текст
  • <samp>: вывод программы
  • <script>: клиентский сценарий
  • <section>: секция
  • <select>: выпадающий список
  • <small>: уменьшенный текст
  • <source>: источник для media элементов
  • <span>: строчный контейнер
  • <strong>: важный текст
  • <style>: встроенные стили
  • <sub>: подстрочный текст
  • <summary>: заголовок элемента details
  • <sup>: надстрочный текст
  • <svg>: векторная графика

  • <td>: ячейка таблицы
  • <template>: контейнер для контента
  • <textarea>: многострочное поле ввода
  • <tfoot>: подвал таблицы
  • <th>: заголовочная ячейка
  • <thead>: заголовок таблицы
  • <time>: дата/время
  • <title>: заголовок документа
  • <tr>: строка таблицы
  • <track>: текстовая дорожка для media элементов
  • U
  • <ul>: ненумерованный список
  • V
  • <var>: переменная
  • <video>: видео
  • W
  • <wbr>: возможный перенос строки

CSS

  • element: селектор по тегу
  • .class: селектор по классу
  • #id: селектор по идентификатору
  • *: универсальный селектор
  • element1, element2: групповой селектор
  • element element: селектор потомков
  • element > element: селектор прямых потомков
  • element + element: смежный селектор
  • element ~ element: селектор братьев
  • :pseudo-class: псевдокласс
  • ::pseudo-element: псевдоэлемент
